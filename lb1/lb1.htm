Програмування

Нижче наведено приклад простої програми для обчислення площі прямокутника мовою Python:

def rectangle_area(width, height):
    return width * height
print("Обчислення площі прямокутника")
w = float(input("Введіть ширину: "))
h = float(input("Введіть висоту: "))
area = rectangle_area(w, h)
print("Площа прямокутника = ", area)
